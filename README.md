# ⚖️ CrossEx-AI

AI Cross Examination Trainer for Law Students.

This tool helps students practice litigation skills using Socratic AI.

Features

- AI witness simulation
- Socratic questioning
- Cross examination training
- Improves legal questioning skills
- Uses OpenAI API

Tech Stack

HTML  
CSS  
JavaScript  
OpenAI API

How to Run

1 Clone the repo

2 Add your OpenAI API key in script.js

3 Open index.html
